# Medicaid Work Requirements and Hospital Readmissions:
## A Difference-in-Differences Analysis in Arkansas

### Overview

This repository contains the Stata code and accompanying paper for an empirical analysis examining the effects of Arkansas's 2018 Medicaid work requirements on hospital Expected Readmission Ratios (ERR).

Using a Difference-in-Differences (DiD) framework, the analysis compares hospitals in Arkansas with hospitals in Mississippi, Missouri, and Oklahoma using publicly available CMS data from 2014–2024.

### Research Question

Do Medicaid work requirements affect hospital readmission outcomes?

### Methodology

- Difference-in-Differences (DiD)
- Event-study specification
- Hospital-level clustered standard errors
- Condition fixed effects
- Robustness check using grouped pre/post specification

### Data

- Centers for Medicare & Medicaid Services (CMS)
- Hospital Expected Readmission Ratio (ERR) data
- 2014–2024

### Software

- Stata (.do)

### Repository Contents

- `analysis.do` — Complete Stata analysis
- `Journal_Paper.pdf` — Final research paper

### Main Finding

The estimated treatment effects following Arkansas's implementation of Medicaid work requirements were generally small and statistically insignificant. While post-treatment coefficients were consistently negative, the analysis did not provide sufficient evidence that the policy significantly affected hospital readmission outcomes.
